### 3. Installation Steps
1. **Run the Installer:** Open the file you downloaded.
2. **Patch Discord:** Select the version of Discord you have installed (Stable, Canary, or PTB) and click **Install**.
3. **Restart Discord:** Completely close Discord (check your system tray to ensure it's fully closed) and reopen it.
4. **Verify:** Go to your **Discord Settings** (the gear icon). Scroll down the left sidebar until you see the **Vencord** section.

## üöÄ My Recommended Plugins
These are the plugins I currently have enabled in my setup. You can find these by searching in **Settings > Vencord > Plugins**.

| Plugin Name | Description |
| :--- | :--- |
| **FakeNitro** | Unlocks Nitro-only features: send custom emojis/stickers and stream in high quality for free. |
| **FakeProfileThemes** | Allows you to use custom profile colors and themes without Nitro. |
| **Decor** | Lets you use and see custom avatar decorations (profile borders). |
| **SilentTyping** | Completely hides the "is typing..." indicator so others can't see when you are writing. |
| **VoiceMessages** | Adds the ability to send voice messages from your PC/Browser, which is normally mobile-only. |
| **NoTrack** | Disables Discord's background telemetry and tracking for better privacy. |
| **NoBlockedMessages** | Fully removes the "Blocked Message" placeholder so you never see messages from blocked users. |
| **BetterGifAltText** | Improves the way GIF descriptions are handled for a better browsing experience. |
| **iLoveSpam** | A utility plugin for managing specific chat interactions. |

## üé® Custom Themes
To make Discord look better:
1. Go to **Vencord > Themes**.
2. Visit [BetterDiscord Themes](https://betterdiscord.app/themes) to find a style you like.
3. Copy the theme's direct link and paste it into the **Theme Links** box in your settings.

## ‚ùì FAQ
**Q: Will I get banned?** A: It is very unlikely. Discord generally does not ban users for client mods unless they are used for malicious purposes.

**Q: Does it work on Mobile?** A: No, Vencord is for Desktop (Windows, Mac, Linux) and Web Browsers only.

**Q: How do I update it?** A: Vencord usually updates itself. If it breaks after a Discord update, just run the installer again and click **Repair**.
